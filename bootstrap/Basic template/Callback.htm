Terugbelafspraak standaard

Nu geven we in het script een aantal opties en omzeilen het dialoogvenster. Het is ook mogelijk om de opgegeven waarde als standaard waarde te gebruiken in het standaard dialoogvenster.

Morgen om 19:00 (direct)
Alleen in de avond tussen 18:00 en 22:00 (direct)
Alleen op Donderdagavond (niet aanpasbaar, niet direct)
Over 2 maand om 18:00, minimaal over 1 maand

Terugbelafspraak via keuze in script

Het is ook mogelijk om via een datepicker control een datum te kiezen en deze door te geven aan CallPro. Formaat YYYY-MM-DD Voor meer informatie kijk op Bootstrap Datepicker documentatie
