Direct doorverbinden

Direct doorverbinden (zonder ruggespraak). Dit wordt ook wel een blind-transfer genoemd. Het gesprek wordt direct doorgeschakeld naar het opgegeven nummer. De Agent komt in de Wrapup fase.

Doorverbinden met ruggespraak

Doorverbinden met ruggespraak. Dit wordt ook wel een attended-transfer genoemd. Het actieve gesprek wordt geparkeerd en de Agent belt het opgegeven nummer. Na ruggespraak kan de Agent het gesprek doorverbinden, of afbreken. In het geval van afbreken krijgt de Agent het actieve gesprek terug.
